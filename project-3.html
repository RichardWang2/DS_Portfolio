Project Overview

This is a project I did when I was working as a Data Discovery Consultant in UC Berkeley. I decided to write a blog about SQL, a skill I had extensive experience with, and a blog is a perfect way to share this experience and knowledge with many other people, including the consultees that need SQL for their data science projects.
I went through basics + intermediate topics in SQL, such as SELECT, GROUP, JOIN, SUBQUERIES, EXISTS, etc. I also used example datasets and queries to demonstrate how to use these SQL queries. In the end, I made a quick recap with sample queries for important SQL concepts for exam/interview reviews.
Give it a read, and I hope you enjoy!
